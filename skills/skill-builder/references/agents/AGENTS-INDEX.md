# Agents Index

> Agents are execution personas. Use them with the skill system, not instead of it.

## How to Use Agents

1. Infer the real job behind the user prompt
2. Pick the smallest suitable agent branch
3. Read one specific agent note when a role clearly matches
4. Use skills, stacks, playbooks, references, and project memory as the source of truth
